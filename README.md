# Apostrophe — React Modal Form Task ✨
This project is a simple React application built with Vite.

# Tech Stack :
* React
* Vite
* TailwindCSS

## How to Run the Project
1. Install dependencies
   
```bash 
npm install
```
2. Start development server
```bash 
npm run dev
```

## Project Structure
```css
src/
├── App.jsx
├── main.jsx
├── index.css
└── components/
    └── FormModal.jsx
```
